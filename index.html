一种类似深度优先算法的全排列

发表于 2018-11-28 | 分类于 算法 | | 阅读次数:

此题来源于《啊哈！算法》第四章第一小节

问题描述

输入一个数n，输出1~n的全排列。

问题分析

理解深度优先算法的关键在于解决“当下该如何做”。至于“下一步怎么做”则与“当下该如何做”是一样的。

比如，我们这里写的dfs(step)函数就是为了解决在第step个数应该填入什么。通常的方法就是把每种尝试都去尝试一遍(通常使用的是for循环，例如此题)。当前这一步解决之后就可以进入到下一步dfs(Step+1)了。而下一步的解决方法和当前这一步的解决方法是完全一样的。下面的的伪代码就是深度优先算法的基本模型(DFS)。

Code

#include <stdio.h>

int a[10],book[10],n;  //C语言的全局变量在没有赋值以前默认为0，因此这里的book数组无需全部再次赋初始值0 

void dfs(int step) //step表示当前需要尝试的位
{
    int i;
    if(step == n+1) //如果是第n+1位，则将前n位的输出
    {
        //输出一组全排列
        for(i=1; i<=n; i++)
        {
            printf("%d", a[i]);
        } 
        printf("\n");        
        return;   //返回之前一步，即最近一次调用dfs函数的地方，然后进行下一次尝试。 
    } 

    for(i=1; i<=n; i++) //从头到尾的尝试 
    {
        if(book[i]==0) //book[i]为0表示i号仍然在手中 
        {
            a[step]=i; //将i号放入第step位中 
            book[i]=1; //将book[i]设为1，表示i号已经不在手中

            dfs(step+1); //进入到下一个盒子的尝试，即是第step+1位

            book[i]=0; //尝试完成，将刚刚尝试的数字收回。进行下一次这一位的尝试。 
        }
    } 
    return;
} 

int main(int argc, char *argv[]) {
    //输入：n为 0~9之间的整数；
    scanf("%d", &n); 
    dfs(1); //首先从第一位开始
    getchar();
    getchar();
    return 0;
}
