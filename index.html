Windows 64位系统下的汇编语言环境搭建

发表于 2018-08-12 | 分类于 汇编 | | 阅读次数:

一、前言

在学习王爽教授的《汇编语言》的时候，我们需要搭建一个汇编语言的运行环境。但是他的书本是以Windows 2000为运行环境的，早已过时。我们现在的大部分人使用的操作系统都是64位的 Win7、Win8或者Win10，64位Windows系统不再支持在命令提示符窗口下进入debug调试模式。同时我们还需要一个masm汇编编译器。所以今天，我将带领你在“现代”操作系统上搭建一个汇编环境。
软件需求：
1.DOSBox 0.74版 （DOSBox是一个模拟软件，没有这个软件，Debug.exe和masm编译器也就无法运行。）
2.Debug.exe
3.masm5.0汇编编译器
